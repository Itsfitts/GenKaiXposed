## Build Instructions

### Prerequisites

- Android Studio Canary (latest)
- JDK 25 (or 24 fallback)
- Gradle 9.4.0-milestone-2
- Android SDK 36 (compileSdk)
- minSdk 34 (Android 14)

### Build Steps

```bash
# Clone repository
git clone https://gitlab.com/a.u.r.a.k.a.i1/Genkai.git
cd Genkai

# Clean build
./gradlew clean

# Assemble debug APK
./gradlew :app:assembleDebug

# Run on device
./gradlew :app:installDebug
```
